# Express API Boilerplate


Stack:
- Node.js + Express
- MySQL (mysql2/promise)
- JWT authentication
- ESLint (eslint:recommended) + Prettier
- Morgan, Helmet, Rate-limit, CORS, lodash


## Setup
1. Copy files
2. `npm install`
4. Buat file `.env` berdasarkan `.env.example`
5. `npm run dev`

## Database
- Design database dapat diakses melalui folder Database
